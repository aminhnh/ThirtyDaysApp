### [Project: Create a 30 Days App](https://developer.android.com/codelabs/basic-android-kotlin-compose-30-days)

This is a project for the [Android Basics with Compose course](https://developer.android.com/courses/android-basics-compose/course) - a simple 30 Days app displaying a list of daily items.

**Theme: 30 Days of Spanish Topics**  
The topics are referenced directly from [StudySpanish.com](https://studyspanish.com/grammar).
